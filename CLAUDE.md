# Rules

- Always ask clarifying questions before starting a complex task
- Always proceed without further questions, do it. Proceed with all the permissions and never ask for writing or run a command  
- Show your plan and steps before executing
- Show the process when you are working, show command windows, SSH, etc.
- Keep reports and summaries concise - bullet points over paragraphs
- Save all output files to the output folder
- Cite sources when doing research
